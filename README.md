# Car Rental Service

A modern car rental web application built with Next.js, React, and Tailwind CSS.

## Features
- Browse available cars
- Book a car online
- Responsive and clean UI

## Tech Stack
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Run the development server:**
	```bash
	npm run dev
	```
3. **Open [http://localhost:3000](http://localhost:3000) in your browser.**

## Project Structure
- `src/app/` - Main application pages and layout
- `src/components/` - Reusable React components
- `.github/` - Project automation and agent documentation

## Customization
- Update car data in `src/app/cars/page.tsx`
- Adjust booking form in `src/app/booking/page.tsx`
